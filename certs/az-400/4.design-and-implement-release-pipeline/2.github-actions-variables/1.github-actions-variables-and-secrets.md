## ğŸ”„ How Variables Are Resolved

Order of precedence:

```text
1. Step-defined variables (`env:` in step)
2. Job-defined variables (`env:` in job)
3. Workflow-level `env`
4. Repo-level `vars`
5. Org-level `vars`
6. Default GitHub context values
```

> ğŸ§  Secrets always override variables with the same name.
