#### Install Self-Hosted Agent

1. Go to Azure DevOps > Org Settings > Agent Pools
2. Create a pool (if needed)
3. Download the agent
4. Run setup script:

   ```bash
   ./config.sh --url https://dev.azure.com/myorg \
               --auth pat \
               --token <PAT_TOKEN> \
               --pool MyCustomPool
   ```

5. Run the agent:

```bash
./run.sh
```
