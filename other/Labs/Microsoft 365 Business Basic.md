# Microsoft 365 Business Basic

**Microsoft 365 Business Basic is licensed per user.**

It supports up to:

```text
300 users
```

but you don't automatically get 300 mailboxes.

---

## Example

Suppose you buy:

```text
Microsoft 365 Business Basic
Quantity: 1 license
```

Then you can create:

```text
hady@hadywafa.com
admin@hadywafa.com
dev@hadywafa.com
hr@hadywafa.com
```

as Entra users.

But only the user assigned the license gets:

```text
✓ Mailbox
✓ Outlook
✓ Teams
✓ OneDrive
```

For example:

```text
hady@hadywafa.com  ← Licensed
admin@hadywafa.com ← Unlicensed
dev@hadywafa.com   ← Unlicensed
```

Result:

```text
hady@hadywafa.com  → Has mailbox
admin@hadywafa.com → No mailbox
dev@hadywafa.com   → No mailbox
```

## For your personal setup

You don't need multiple licensed users.

The best design is:

### One licensed user

```text
hady@hadywafa.com
```

### Multiple aliases

```text
career@hadywafa.com

aws@hadywafa.com

```

All aliases deliver into:

```text
hady@hadywafa.com
```

same inbox.
